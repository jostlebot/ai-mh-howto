Getting Started with Claude

What is Claude?

Claude is an AI assistant made by Anthropic. Think of it like ChatGPT's thoughtful cousin — it's particularly good at nuanced conversations, writing, and helping you build things.

There are a few ways to use Claude:

  • Claude.ai — The website where you chat with Claude (like ChatGPT's interface)
  • Claude Code — A tool that lets Claude work directly on your computer, reading and writing files
  • The API — For building Claude into your own applications

Step 1: Create an Anthropic Account

  1. Go to claude.ai
  2. Click "Sign up" and create your account
  3. Verify your email

That's it — you can now chat with Claude on the web.

Step 2: Install Claude Code (Optional but Powerful)

Claude Code lets Claude work on files on your computer. It's like having a collaborator who can actually edit your documents and projects.

First, you need Node.js installed. If you've never done this:
  1. Go to nodejs.org
  2. Download the "LTS" version (the button on the left)
  3. Run the installer, click through the steps

Now install Claude Code:

  1. Open your Terminal (on Mac: search "Terminal" in Spotlight)
  2. Copy and paste this command, then press Enter:
npm install -g @anthropic-ai/claude-code

Wait for it to finish. You'll see some text scroll by — that's normal.

Step 3: Start Claude Code

In your Terminal, type:

claude

The first time, it will ask you to log in with your Anthropic account. Follow the prompts.

Once logged in, you can start talking to Claude right in your terminal. Try typing:

What can you help me with?

Step 4: Your First Real Task

Let's have Claude create something. In the terminal where Claude is running, try:

Create a simple HTML page that says "Hello, I'm building with AI!" and save it to my Desktop

Claude will create the file for you. Open it in your browser to see your creation.

Key Things to Know

  • Claude can see your files — When you're in a folder and run Claude Code, it can read and edit files there
  • Be specific — The clearer your instructions, the better the results
  • You're in control — Claude will ask permission before making changes
  • It's conversational — You can say "actually, change that to..." just like talking to a person
